## üéì About This Repository

This repository serves as a **preparation hub for Polytechnic students** appearing for their Java Programming semester final exams. It contains solved practical problems, standard algorithms, and core Object-Oriented Programming (OOP) examples often asked in board exams and lab finals.

I have uploaded these codes to help my fellow students understand the logic and syntax required to score well in the practical and theory sections.

## üìÇ Study Topics & File Guide

### üîπ Module 1: Object-Oriented Programming (OOP)
Essential concepts for the written and practical exams.

| Concept | File Name | Description |
| :--- | :--- | :--- |
| **Inheritance** | `Multilevel_Inheritance.java` | Example of Multilevel Inheritance (First -> Second -> Third). |
| **Polymorphism** | `Overloading.java` | Compile-time polymorphism showing method overloading (adding 2 vs 3 numbers). |
